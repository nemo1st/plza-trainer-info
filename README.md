## Dependencies
- Python 3.13

## How to Use it

```sh
# Repair save file
$ python3 main.py main

## Display CoreData
python3 main.py --core-data main

## Modify name and ID (with leading zeros)
python3 main.py --modify main --name ZA --id 0810123456
```
